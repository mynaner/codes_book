### 创建vite初始化项目 

> ```cmd
> npm create vite@latest
> // or
> yarn create vite
> ```

> ```
> npm install vue-router@4  // 路由
> ```
>
> ```
> npm install pinia // pinia 数据管理
> ```
>
> ```
> npm install axios // http请求
> ```



```js

npm i -D naive-ui
```
